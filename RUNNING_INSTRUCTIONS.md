# Running WebGame,
,
To run the WebGame project locally, follow these steps:,
,
## Frontend (Client),
,
1. **Navigate to the client directory**:,
   ```bash,
   cd client,
   ```,
2. **Install dependencies**:,
   ```bash,
   npm install,
   ```,
3. **Start the development server**:,
   ```bash,
   npm run dev,
   ```,
4. **Access the game**:,
   Open the local URL displayed in your terminal (usually `http://localhost:5173`).,
,
## Backend (Server),
,
*Note: The backend is currently in the initial setup phase and files are placeholders.*,
,
1. **Navigate to the server directory**:,
   ```bash,
   cd server,
   ```,
2. **Install dependencies**:,
   ```bash,
   npm install,
   ```,
3. **Start the server**:,
   ```bash,
   node index.js,
   ```
